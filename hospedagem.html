O que é?

Resumidamente é um serviço que permite armazenar e disponibilizar um site na internet.

O que é necessário para hospedar um site?

Para hospedar um site é necessário você possuir um dominio que pode ser adiquirido de diversas formas, um servidor para realizar a hospedagem e por fim mas não menos importante o código do site que deseja hospedar.

Locais onde você pode realizar a hospedagem:

Existem diversos locais onde você pode realizar a hospedagem de um site, os mais conhecidos são: Amazon Web services (AWS), HostGator e Hostinger.

Custos:

Os custos envolvidos podem variar bastante pois você pode hospedar na sua própria máquina o que torna gratuito porém não é tão seguro, você também pode hospedar em alguns locais que oferecem a hospedagem gratuita, como por exemplo o próprio GitHub com o GitHub Pages, também vai ter gastos envolvendo o dominio para que seu site possa ser hospedado, porém alguns sistemas de hospedagem lhe oferecem um dominio gratuito caso você compre um pacote de hospedagem X.
Abaixo vou estar deixando algumas opções de sites para dominio e hospedagem e seus respectivos preços encontrados:

Hospedagem:

  • AWS: Seu custo inicial é gratuito por um ano, porém após este 1 ano ele passa a ser cobrado de acordo com seu uso, portante o preço especifico é variavel.
  • Hostinger: O Hostinger possui diversos pacotes que podem ir desde 29,99 (sem descontos) até 99,99(sem descontos), este valor deve ser pago mensalmente. Os valores informados podem variar bastante pois possuem diversos descontos e a partir do segundo pacote você ganha 3 meses gratuitos de uso do serviço de hospedagem e também um dominio gratuito.
  • HostGator: O HostGator é outra empresa que possui diversos pacotes que podem atender à diversas necessidades, mas de maneira geral os preços vão de 11,29(sem descontos) até 28,91(sem descontos) por mês, porém é bom analisar cada uma das opções antes para ver qual se adequa melhor ao seu tipo de site.
  • GitHub Pages: Por fim temos o GitHub Pages, de todos os informados anteriormente ele é o único que possui o serviço de hospedagem de maneira grauita, porém ele só hospeda sites estáticos (e com bastante esforço caso seja sua primeira vez um site em react também), pelo GitHub Pages você pode também mudaro dominio de seu site, porém vai ser necessário comprar um antes.

Dominios:

DNS:

Resumidamente se você estiver usando serviços de hospedagem ou outros provedores de serviços diferentes, precisará atualizar os servidores de nomes do seu domínio para apontar para os servidores corretos. Isso permite que o tráfego seja direcionado ao local correto onde seu site está hospedado.
